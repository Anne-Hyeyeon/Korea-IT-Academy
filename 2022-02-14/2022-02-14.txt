
- HTML (Hyper Text Markup Language)

- 웹(WWW) - World Wide Web의 약어, 여기서 Web은 그물(거미줄)이다.
- 전 세계 사람들이 정보를 공유할 수 있는 사이버 공간
- 웹 콘텐츠를 이용하기 위해서는 '웹 브라우저'가 필요.
- 웹 서버 : 웹 서비스를 제공하는 컴퓨터의 하드웨어 or 소프트웨어
- 클라이언트 : <->웹 서버, 서비스를 이용하는 컴퓨터 or 사용자
쉽게 얘기하면 우리집 컴퓨터가 클라이언트, 정보를 가지고 있는 공간이 서버
서버가 폭발했다? 너무 많은 클라이언트가 동시에 접속하여 서버가 멈췄다는 뜻
(보통 이러면 서버를 재부팅한다고 함ㅎㅎ)
- 웹 호스팅 : 서버상 계정과 디스크 공간을 임대해줌으로서 홈페이지를 구축할 수 있게 함
(온라인 임대사업이라고 할 수 있다.)
- 웹페이지 : 브라우저 화면에서 보이는 화면 <- HTML, CSS, JavaScript 등의 소스 파일 + 데이터로 구성
- 홈페이지 : 여러 개의 웹페이지 중 주로 첫 페이지를 뜻함

- HTML : 웹브라우저가 알아들을 수 있는 언어
- 사용하는 텍스트 에디터 : 비주얼 스튜디오 코드
