[스위치 조건문]

참일때, 거짓일때, 이도저도 아닐 때 (default) 로 나누어서 실행!

if문이랑 다른 점 : if문은 조건에 따라 순서가 나오고, switch는 조건에 따라 케이스에 있는 값이 바로 실행된다.

switch (비교문) {
  switch (input % 2)
    case 0:
      alert('짝수입니다.');
      break;
    case 1:
      alert('홀수입니다.');
      break;
    default:
      alert('숫자가 아닙니다.');
      break
  
}

[상항 연산자]
- 연산자이지만 프로그램의 진행이 조건에 따라 변화 가능
- 기본 형태 
(불리언 표현식)?(참일 때 실행하는 문장)(거짓일 때 실행하는 문장)
- 한 줄로 표시할 수 있을 때만 사용
- 코드 길이 줄일 수 있음


- 논리합 연산자를 사용한 표현식은 뒤에 어떤 값이 들어가도 참
-> 자바스크립트의 논리합 연사자는 화법이 참이면 우변ㅇ르 실행하지 ㅇ낳음
