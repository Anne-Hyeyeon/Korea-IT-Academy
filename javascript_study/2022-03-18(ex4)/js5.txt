[while 반복문]
- 조건이 변화하지 않는다면 반복문 안을 무한히 반복
- 조건을 거짓으로 만들 수 있는 내용이 문장 안에 포함돼야 함.

while (불리언 표현식) {
  문장
}

[do while 반복문]
- 조건이 참인지 거짓인지와 상관없이, 내부의 문장을 최소한 한 번을 실행해야 하는 경우 사용
- 조건을 비교하는 부분이 가장 마지막에 위치

do {
  문장
} while (불리언 표현식);


[for 반복문]
while 반복문은 조건에 비중이 있다.
but, 조건보다 횟수에 비중을 둘 때 for 반복문 사용

for (초기식; 조건식; 종결식) {
  문장
}
